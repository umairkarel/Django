## Django Blog
A Functional Blog-App built with Django. 

---
### Features:
- User
    - Register
    - Login
    - Update Profile
- Post
    - Create
    - Update
    - Delete
---
### Developer Setup:
- Fork this repository.
- Verify if the Django project is working by running the development server.
```bash
python manage.py migrate
```
```bash
python manage.py runserver
```
